Accordion

An accordion is a vertically stacked list of headers that reveal or hide related sections of content.

Note: For now, Accordion uses Angular animations. As no other components use the package it is not bundled with Kirby for performance reasons. As a result it is necessary to have @angular/platform-browser installed and import BrowserAnimationsModule yourself when using the accordion.
