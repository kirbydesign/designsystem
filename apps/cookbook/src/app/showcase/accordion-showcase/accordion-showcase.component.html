An accordion is a vertically stacked list of headers that reveal or hide related sections of content.

Setup

Accordions uses Angular animations for animating between its collapsed and expanded state. As no other components depend on the package it is not bundled with Kirby for performance reasons and should instead be provided in the application as detailed in the Angular docs for enabling the animations module .

Simple accordion

With initially expanded item

It is possible to have an accordion expanded initially to set focus on a specific information.

Accordion inside a card

Accordions may be placed within a card without further adjustments needed to make it stand out from the canvas.

Accordion with list

Placing a list within an accordion item will automatically adapt spacing and strip the necessary list-styling to ensure that the list looks correct without further adjustments.

Additional styling

Similarly to the simple accordion, the accordion with a list can also be wrapped in a card to make it stand out from the canvas.

Accordion toggle event

A toggle event is emitted whenever an accordion is expanded or collapsed allowing for further interactions.
