Accordion

An accordion is a vertically stacked list of headers that reveal or hide related sections of content.

Note: Accordion uses Angular animations for animating between its collapsed and expanded state. As no other components depend on the package it is not bundled with Kirby for performance reasons and should instead be provided in the application as detailed in the Angular docs for enabling the animations module .
