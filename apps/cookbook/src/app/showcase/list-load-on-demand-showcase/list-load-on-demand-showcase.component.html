The kirby list will emit at loadMore event, when the user has scrolled 80% of the list. It is the responsibility of the consumer to add more items to the bound items.

The event emitted is of type KirbyLoadMoreEvent which exposes a complete callback function.

The complete callback function should be called when the loading of the additional items are done.

If complete is called with the disableLoadMore argument set to true, no further events will be emitted. Further more, if any is given, the noMoreItemsText will be shown.
