Note: the default behaviour and styling of the stock chart might change in the near future as work is still being done to enhance it. More details can be found on this issue in the Github repository.

The stock chart visualizes the price of a security over time where time is along the x-axis and the price is on the y-axis.

The kirby-stock-chart element shares a lot of its API with the kirby-chart element. For more info and examples see: the chart showcase .

The labels of the x-axis show the day and month of the datapoints. If you need a different increment, ie. hours or years you should provide your own labels through the `labels` property.

Please note that this chart only accepts data where each datapoint consists of a value for both x and y. 

  {{ '{' }}
    x: 1231831920, // Time as epoch timestamp
    y: 10 // Value at time 'x'
  {{ '}' }}

Stock Chart with multiple datasets

It is possible to show and compare the price of multiple securities in one chart. The ChartDataLabelOptions property valueSuffix can be useful if for example data comes in percent. By setting valueSuffix to the string '%' values in the data labels will be suffixed with '%' as can be seen on the below example.
