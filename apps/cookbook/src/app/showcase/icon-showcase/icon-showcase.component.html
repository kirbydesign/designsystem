Performance:

A bonus with the web component is that it uses Intersection Observer to only request icons which are viewable by the users (icons above the fold). The component will dynamically load an SVG for each icon, so your app is only requesting the icons that you need.

Also note that only visible icons are loaded, and icons which are "below the fold" and hidden from the user's view do not make fetch requests for the svg resource. Read more
