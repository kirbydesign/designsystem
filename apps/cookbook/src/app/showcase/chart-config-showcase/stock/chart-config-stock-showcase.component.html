Stock Chart built from configs

This page describes how to implement various types of charts using the Kirby standard configurations. If you would like a guide on how to get started, go to this guide.

The philosophy of the Kirby chart configs is to give the implementer full control over the chart, while providing standard configs, which makes it easy to create custom charts which comply with the Kirby design.

The idea behind Kirby chart config is only to provide configurations and not the canvas itself. It is for the implementer to provide the canvas because it gives the implementer ownership.
