Stock Chart build from configs

This page describes how to implement charts using Kirby standart configurations

The philisophy of the Kirby chart configs is to give the implementer full control over the chart, while providing standart configs which makes it easier to create custom charts which complies with the Kirby design.

Basic stock chart

The idea behind Kirby chart config is to only provide configurations and not the canvas itself. It is for the implementer to provide the canvas, because it gives the implementer ownership.
