Stock Chart build from configs

This page describes how to implement charts using Kirby standart configurations. If you would like a guide on how to get started, go to this guide

The philisophy of the Kirby chart configs is to give the implementer full control over the chart, while providing standart configs which makes it easy to create custom charts which complies with the Kirby design.

The idea behind Kirby chart config is to only provide configurations and not the canvas itself. It is for the implementer to provide the canvas, because it gives the implementer ownership.
