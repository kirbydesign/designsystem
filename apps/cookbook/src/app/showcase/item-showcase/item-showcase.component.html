The kirby-item component consists of three possible columns: the default , which spans the entire item, and optionally the 'start' and 'end' columns (aka 'slots' )
In addition to this there is a special slot for flags named 'outside' .

The default column is the content that isn't slotted into either 'start' or 'end' .
If there are more than one line of content it can be contained within a kirby-label that by default stacks content vertically (multi lines).
Optionally it can be configured to stack horizontally: <kirby-label direction="horizontal">...</kirby-label>

Please note , if you need a label to span a title and a value, you can wrap the title and value in another kirby-label as in this example

The start column is usually only used with avatars, icons and such. To place content in the start column specify the slot="start" attribute on your content. If no content is placed into the 'start' slot, the first column will not be shown.

The end column is usually used for values such as number, currency, date etc. To place content in the end column specify the slot="end" attribute on your content. If multiple lines are needed the values can be wrapped in a kirby-label as in this example .
Values can optionally be contained in a time or data tag as in this example .

The outside slot can be used for flags as in this example .

Rotation animation

It is possible to trigger an animation that rotates the disclosure icon by passing a boolean flag to [rotateIcon] . When set to true it will rotate the icon 180 degrees and when set to false it will rotate the icon back to it's original position.

Please note: the animation of the disclosure icon is only available for: 'arrow-down' | 'arrow-up'
