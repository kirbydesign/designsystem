Slides are a collection of cards you can navigate horizontally in, swipe on touch devices and click navigation on desktop.

Usage

Slides are great to gather and contain content without using a lot of space, but the content has to be closely related.
Only use slides if you have 3 or more cards. If you have a lot of content, or if it takes more than four interactions (swipe or clicks), consider using a list format instead.
Slides are best suited for inspirationel content where users are expected to be in a explorative mode.

*Please note: It is possible to set [simpleSlider]="true" if you want to revert back to the old kirby-slides design.
