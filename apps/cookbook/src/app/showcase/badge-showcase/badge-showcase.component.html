Badges are inlines block elements that often are used on top of or close to another component. They can be used with numbers, text and icons. A small badge version also exists which does not support slotted content but can be used to indicate status.

Number

Using a badge with an number can be a good way to indicate status on an element.

Text

Badges can also be used together with a text. Flags might in some cases be a better alternative.

Icon

It is possible to slot an icon to display it in a badge. This might be used to indicate the existence of an associated element.


Small Badge

Setting the input property size to 'sm' will display a small badge without any content. This can be useful to indicate status on an element.
