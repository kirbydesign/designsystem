Tabs

Tabs are a top level navigation component to implement a tab-based navigation. The component is a container of individual content.

Icons

An icon for the tab can be specified by nesting a kirby-icon inside the kirby-tab-button tag.

Alternative icon for selected tab

If you need a different icon for the selected state of the tab, you can specify an additional icon with the attribute: selected-tab (see example below).

Router integration:

When used with Angular's router the routerLink property of the kirby-tab-button should be a reference to the route path.

Content of a tab will automatically be included by a router-outlet . Therefore the route specified by the tab's routerLink property should be a child route of the route hosting your "tabs page" component.
