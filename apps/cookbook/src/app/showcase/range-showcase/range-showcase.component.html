The Range slider lets users select from a range of values by moving the slider knob.

Simple

Step (ticks)

Tick marks are displayed based on the step value. If the amount of ticks exceeds 10 , the step property will automatically be adjusted.

Provided values that doesn't match any tick points, will automatically be changed to the nearest tick point.

Pin - the slider value is shown when the knob is pressed

Additionally, the pinFormatter function can be supplied to customize the formatting of the range value shown to the user in the pin. It defaults to the range slider's integer-value.
