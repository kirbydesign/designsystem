Header is a component that manages the layout of the header section of kirby-page. It is responsive and will change the layout according to the device size. It allows for a wide range of use cases including support for avatar, flags etc.

Default

Flag

A flag can be used to indicate a status. To use flag you simply add kirby-flag in the header and that should position the flag correctly and manage spacing.

Key value

In some cases a number is the primary information on a page. When that's the case you should use the `value` prop in addition to the `title` prop. This will highlight the value as a emphasized number on the page and make the title secondary as shown in the example below:

Avatar

A header can be used together with a Avatar. You simply add kirby-avatar in the header and that should position the avatar correctly and manage spacing.
