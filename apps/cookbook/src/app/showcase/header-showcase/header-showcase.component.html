Header is a component that lays out the header section of kirby-page. It is responsive and will change the layout according to the device size. The header allows for a wide range of customizability but you should follow the UX guidelines for header to provide the best expereience.

Default

Flag

A flag can be used to indicate a status. To use flag you simply add kirby-flag in the header and add the *kirbyHeaderFlag directive to it and that should position the flag correctly and manage spacing.

Key-value

In some cases a number is the primary information on a page. When that's the case you should use value adjust the header accordingly. To use value you simply add ng-container or another element with the value you want to display in the header and add the *kirbyHeaderValue directive to it and that should resize the title and display the value provided below.

Actions

As an alternative to toolbar actions the header also allows you to insert actions in the header itself. To insert actions simply add a ng-container or another element with the actions you want to display and add the *kirbyHeaderActions directive to it.

Avatar

A header can be used together with a Avatar. You simply add a kirby-avatar in the header and add the *kirbyHeaderAvatar directive to it and that should position the avatar correctly and manage spacing.
