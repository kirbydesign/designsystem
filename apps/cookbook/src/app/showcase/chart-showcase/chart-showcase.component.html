Properties:

Name Description Input values
height The height of the chart in px for web and dp for native number
type The type of chart donut | pie | areaspline
showDataLabels Enable or disable labels around pie/donut charts (default: true) true, false
description Description of chart for accessibility string - e.g. "Distribution of assets in custody account"
data Data input for chart See examples above for exact format
options The Highcharts options to configure the chart with (will override existing options using a deep merge) See Highcharts options docs here
