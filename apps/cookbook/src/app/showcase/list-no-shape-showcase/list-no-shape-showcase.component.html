List with no background shape:

Setting shape="none" on a Kirby List removes the background and shadow from the list and the list item containers to provide flexibility when combined with other layout containers.
Dividers will still be shown between list items, these can be removed by setting [showDividers]="false".

Simple list with no shape

Detailed card with list no shape

The list with no shape can be used when the list is inside another layout component, that comes with it's own shape or background, e.g. within a card.

Multi card list (having no shape)

When using the list with a custom item template where each item comes with it's own shape or background, e.g. using a card for each item, set both shape="none" and [showDivider]="false" to omit any background, shadow and dividers from the list.
