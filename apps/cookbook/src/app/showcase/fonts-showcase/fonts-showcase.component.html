The font sizes can be set by either using the font classes, e.g. kirby-text-large , or by using the right tags like h1, h2 & h3

Font can also be set with the font-size function directly on your own classes, e.g:

  .my-own-class {{ '{' }}
    font-size: font-size("l");
    ...
  {{ '}' }}

Display 1

font-size: font-size("xxxxl"); font-weight: font-weight("black"); CLASS: kirby-text-display-1

Display 2

font-size: font-size("xxxl"); font-weight: font-weight("black"); CLASS: kirby-text-display-2

Display 3

font-size: font-size("xxl"); font-weight: font-weight("black"); CLASS: kirby-text-display-3

h1 - Heading 1

font-size: font-size("xl"); font-weight: font-weight("black"); CLASS: h1 or .kirby-text-xlarge

h2 - Heading 2

font-size: font-size("l"); font-weight: font-weight("bold"); CLASS: h2 or kirby-text-large

h3 - Heading 3

font-size: font-size("m"); font-weight: font-weight("bold"); CLASS: h3 or kirby-text-medium

h4 - Section header

font-size: font-size("n"); font-weight: font-weight("bold"); CLASS: h4

p - Paragraph (Default body font)

font-size: font-size("n"); CLASS: kirby-text-normal

Small text

font-size: font-size("s"); CLASS: kirby-text-small

Small text - light

font-size: font-size("s"); font-weight: font-weight("light"); CLASS: kirby-text-small-light

x-small text

font-size: font-size("xs"); CLASS: kirby-text-xsmall

xx-small text

font-size: font-size("xxs"); CLASS: kirby-text-xxsmall
