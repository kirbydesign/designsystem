Toggle is an input control for binary options, commonly used for managing choices and switches.

Examples

Angular Forms

The toggle can be used directly in template driven and reactive forms . To use toggle in a reactive form, bind your form control to <kirby-toggle> with simple data binding:

<kirby-toggle [formControl]="yourFormControl"> // or simply: <kirby-toggle formControlName="yourFormControlName">

Accessibility

Always provide a label for the toggle to inform screen reader users what the toggle is controlling.

When using the toggle without a visible label, set the aria-label property instead.
