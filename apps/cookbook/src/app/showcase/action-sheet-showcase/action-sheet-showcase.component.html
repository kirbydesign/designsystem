To show the action sheet, create an ActionSheetConfig and pass it to modalController.showActionSheet . Each item must to be of type ActionSheetItem . const config: ActionSheetConfig={{ '{' }} header: 'Your action sheet header', subheader: 'Your action sheet sub-header', items: [ {{ '{' }} id: '1', text: 'Option 1' {{ '}' }}, {{ '{' }} id: '2', text: 'Option 2' {{ '}' }}, {{ '{' }} id: '3', text: 'Option 3' {{ '}' }}, ], cancelButtonText: 'Custom cancel', {{ '}' }}; this.modalController.showActionSheet(config); (Optional) if you need to obtain data back from the action sheet, you can pass a callback function: this.modalController.showActionSheet(config, onItemSelect); where onItemSelect can be e.g.: onItemSelect(item: ActionSheetItem) {{ '{' }}...{{ '}' }} We can use modalController.hideTopmost to hide (and destroy its component) the topmost action sheet. We can pass an optional result parameter to it, which is then going to be available inside the callback
