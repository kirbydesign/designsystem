Data Tables allow the consumer to show large amounts of complex data in tabular form, with rows and columns. Columns can be named through the use of headers.

Please note: Kirby data table is very modular. This means that it lacks a number of features and all uses should be thoroughly aligned with a designer before use (use the corresponding design guidelines).

Prominent features that are missing include:
  • Built in responsivity (i.e., no scroll/collapsible columns/etc.)
  • Draggable rown
  • Built in simple sort
