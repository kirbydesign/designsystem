Data Tables allow the consumer to show large amounts of complex data in tabular form, with rows and columns. Columns can be named through the use of headers.

Setup

The Kirby data table is a set of CSS classes that styles the native HTML table element.
Ensure your application includes the global Kirby styles as explained in our readme file .

Simple data table

By default the data table has minimal styling - only the headers are styled and divider lines are added between the rows.

Additional styling and interactions

The data table can be wrapped in a Kirby Card to make it stand out from the canvas.

The rows can be made selectable by applying the kirby-selectable-row CSS class and handling the click() event.

Sortable table

The data table exposes styling and hooks for making the table sortable.

To enable sorting you must import the TableSortableComponent into your Angular module or standalone component.
