Data Tables allow the consumer to show large amounts of complex data in tabular form, with rows and columns. Columns can be named through the use of headers.

Sortable table

Data table wrapped in a card with sorting. In this example the Name , Height & Weight headings are sortable. We recommend to visit the MDN docs on sorting, if you're performing client side sorting.

Please note that the Kirby data table does not do any sorting. The sorting logic must be implemented manually and can be triggered by adding the Angular (click) event listener as shown in the code example above. By adding [sortable]="true" to a th element, it will render the sorting icon (up/down arrow) when the element is clicked.
