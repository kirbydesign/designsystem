An action list is a list of actions that displays in a popup. Action list replaces action-sheet for both mobile and web.

It is up to the user of action-list to add individual kirby-items to get the desired outcome, action-list simply provides the shell of handling the list itself. All functionality that is expected from an item in the list, is up to the user to implement. See examples.

Note: Only kirby-item are allowed as children of an action-list. This is to ensure consistency in design and implementation
