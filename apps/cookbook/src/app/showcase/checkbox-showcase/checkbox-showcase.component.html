A Checkbox is a control that permits the user to select options. This can be a binary choice, i.e. a choice between one of two possible mutually exclusive options. For example, the user may have to answer 'yes' (checked) or 'no' (not checked) on a simple yes/no question. Multiple check boxes in a group may be used to either select multipe items.

Types

There are two types of Checkboxes:

  1. Attention Level 1: Use as a binary choice for a standalone check box
  2. Attention Level 2: Use in a list of check boxes, where multiple choices can be selected

Sizes

The small and medium sizes apply to the clickable/tapable area (highlighted in the examples below) while the checkbox itself remains the same size.

The extra small size applies to both click-area and checkbox, but should be used with caution.

The extra small size is mainly intended for use in expert systems or information dense interfaces where the reduced size is a necessity but note that it comes at the cost of degraded accessibility.

Angular Forms

The checkbox can be used directly in template driven and reactive forms . To use checkbox in a reactive form, bind your form control to <kirby-checkbox> with simple data binding:

<kirby-checkbox [formControl]="yourFormControl"> // or simply: <kirby-checkbox formControlName="yourFormControlName">
