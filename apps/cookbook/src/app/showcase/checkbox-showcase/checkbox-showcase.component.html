A Checkbox is a control that permits the user to select options. This can be a binary choice, i.e. a choice between one of two possible mutually exclusive options. For example, the user may have to answer 'yes' (checked) or 'no' (not checked) on a simple yes/no question. Multiple check boxes in a group may be used to either select multipe items.
