Grid works well for a layout with a known number of columns. The columns can be configured with multiple breakpoints to specify the column span of each child. Breakpoints makes is easy to adapt the layout to different screen sizes.

Breakpoints

Each breakpoint is named using t-shirt sizes and relates to a defined screen size or bigger:

  • xs, extra-small: 0px
  • sm, small: 576px
  • md, medium: 768px
  • lg, large: 992px
  • xl, extra-large: 1200px
