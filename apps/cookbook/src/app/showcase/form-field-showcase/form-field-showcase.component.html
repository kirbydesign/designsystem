Numeric

kirby-input of type number will automatically get a numeric mask (unless you disable it, see notes below).

This adds grouping seperators & restrict the value to be numeric only. It also has addional configuration available for allowing negative values, restricting the value to have an maximum value and more (see "Numeric Input: Specific properties" for more details).

*Please note: Numeric input is based on LOCALE_ID. When DA grouping separator will be . and decimal will be ,

*Please note: mask can be disabled by adding kirby-input-disable-mask

*Please note: If you need to apply the mask for non-kirby inputs, it can be done by using kirby-input-numeric
