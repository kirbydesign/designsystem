Decimal mask

Restricts the input to numbers only. It ensures correct keyboard, adds grouping separators based on locale, and will default to allow two fractional digits. These can be configured by setting [precision]="amount of digits" .

Input can be restricted to a specified range with [min] and [max]

As default, negative numbers are disallowed, however, they can be allowed by setting either [allowMinus]="true" or entering a negative number to [min] [min]="negative number"

Group separators can be disabled with [disableGroupSeperator] .

*Please note: using maxlength will disallow fractional digits.

Date

By default (i.e. when not using the native datepicker) the value supplied to the input should be formatted according to the current locale. When opting into the native datepicker, the value should be in the Date strings format .
