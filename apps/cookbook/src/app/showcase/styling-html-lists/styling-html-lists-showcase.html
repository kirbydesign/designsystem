Kirby offers default styling of ordered <ol> and unordered <ul> HTML lists (but not description/definition lists <dl> ).

There are some good rules of thumb how to use a bulleted list

  • Don’t use a period at the end of an item
  • Don’t use a period at the end of the last bullet in a series of bullets
  • Try to keep the amount of bullets between 3 and 5
  • Try not to use nested lists

How to "opt-out" of Kirby's default HTML list styles

Kirby's default styles for HTML lists are mainly based on the user agent ("browser") styles. In other words, Kirby list styles only change the horizontal "indentation" and the vertical space between list items.

To control the space between list items you’ll want to target list items with sufficiently high specificity, e.g.: 

  ol li,
  ul li {{ '{' }}
    margin-block-start: revert;
  {{ '}' }}

You may also need to adjust the line-height. Which value to use will depend on the specific case, e.g.: 

  ol, ul {{'{' }}
    line-height: normal;
  {{'}' }}

You can control the "indentation" of lists by changing the value of margin-inline-start , e.g. 

  ol, ul {{'{' }}
    margin-inline-start: 1em;
  {{'}' }}
