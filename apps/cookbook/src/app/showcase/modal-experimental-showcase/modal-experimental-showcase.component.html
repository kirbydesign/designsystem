A Modal is a dialog that appears on top of the app's content, and must be dismissed by the app before interaction can resume. It is useful as a select component when there are a lot of options to choose from, or when filtering items in a list, as well as many other use cases.

Modals can be generated programmatically or used as components.

Components

The modal components must be used within a kirby-page or on a page with no scrolling as the modal will automatically apply a class to the body that prevents scrolling. This is not an issue within a kirby-page as that will handle the overflow and not the body.
