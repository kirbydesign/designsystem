Configuration

The floating action button is always shown with a close-icon when in its opened state. In order to show the icon correctly it needs to be configured as described in the project readme .

(Optional) if you need to obtain data back from the fab sheet, you can pass a callback function:

(itemSelect)="onItemSelect($event)" where onItemSelect can be e.g.: onItemSelect(item: ActionSheetItem) {{ '{' }}...{{ '}' }}
