Kirby Extensions for Angular

The Kirby Extensions library is a collection of design guidelines and components that build on top of the core Kirby Design library.

Components in this library are developed and maintained by product teams, with support from Team Kirby .

The components follow the Kirby Design guidelines, but they differ from the core components by being more product-specific. This means that components in this library are shared across one or more products, and might not be universally useful to the whole ecosystem. Though some components can also become core-candidates and graduate once they are battle-tested and initial experimentation is over.

Extensions allow for quicker iteration and less focus on generalising behavior, while the test and especially documentation burden is also significantly lower.

The components are documented on extensions.kirby.design where additional information about installing, using and contributing extensions can be found.
