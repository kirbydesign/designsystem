Inside the parent (caller) component:

To show the modal, create a ModalConfig and pass it to the ModalController.showModal: constructor(private modalController: ModalController, private vcRef: ViewContainerRef) {{ '{}' }} showModal() {{ '{' }} const config: ModalConfig = {{ '{' }} title: 'Your Modal Title', titleHorizontalAlignment: 'center', component: YourEmbeddedModalComponent, {{ '}' }}; this.modalController.showModal(config, this.vcRef); {{ '}' }}

(Optional)
If you need to obtain data back from the modal, you can pass a callback function: this.modalController.showModal(config, this.vcRef, myCallback); myCallback(dataReturnedByModal: any) {{ '{' }} ... {{ '}' }}


Inside the nested component:

To create a nested component, ensure:

  1. the nested component must be declared (module file, or inside your @Component decorator)
  2. the nested component must be listed in your entryComponents (module file)

In order to hide a modal from within your nested component (and destroy its component), you can use: constructor(private modalController: ModalController) {{ '{}' }} this.modalController.hideTopmost();

(Optional)
You can additionally pass data, which will be available in your callback: let returnData = ...; this.modalController.hideTopmost(returnData); It is possible to nest more modals from within your nested components,
by using the same services and principles as above.
