To show the alert, create an AlertConfig and pass it to modalController.showAlert. const config: AlertConfig={{ '{' }} title: 'Your alert title', message: 'Your alert message', okBtnText: string; cancelBtnText: string; {{ '}' }}; this.modalController.showAlert(config); (Optional) if you need to obtain data back from the alert, you can pass a callback function: this.modalController.showAlert(config, myCallback); where myCallback can be e.g.: myCallback(selection: boolean) {{ '{' }} ... {{ '}' }}
