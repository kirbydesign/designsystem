Below is the Kirby color palette. You can try out the different color combinations there and see how it can be used in the CSS in the code example box. All the colors are fetched from the scss/themes/_colors.scss file.

Example use (Click on a color below to change it):
Contrast Text
Tint
Shade
.color-sample-example {{ '{' }}
  color: get-color('{{ selectedOnColor }}');
  background-color: get-color('{{ selectedColor }}');
{{ '}' }}
.color-sample-tint {{ '{' }}
  background-color: get-color('{{ selectedColor }}-tint');
{{ '}' }}
.color-sample-shade {{ '{' }}
  background-color: get-color('{{ selectedColor }}-shade');
{{ '}' }}

The get-color function will output CSS variables for web and output the color values directly for Nativescript.
