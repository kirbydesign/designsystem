Item

Description:

The item-component consists of three possible columns. First column only used with kirby-avatars, kirby-icons and such. Placement happens automatically. If no avatars or icons is present first column will not be show.

Content in Column #2 should always be contained within a kirby-label.

kirby-label stacks content vertically (multi lines)

Values should be contained in a kirby-value (though not sub-values)

Third column content need not be contained in a kirby-label unless multiple lines is needed
