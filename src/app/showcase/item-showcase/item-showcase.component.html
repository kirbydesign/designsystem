The item component consists of three possible columns: the default, which spans the entire item, and optionally the 'start' and 'end' columns (aka 'slots').

The start column is usually only used with avatars, icons and such. Placement happens automatically when used with slot="start". If no content is placed into the 'start' slot, the first column will not be shown.

The default column is usually contained within a kirby-label.
By default kirby-label stacks content vertically (multi lines), but can optinally be configued to stack horizontally:
<kirby-label direction="horizontal">...</kirby-label>

The end column content need not be contained in a kirby-label unless multiple lines is needed. Placement happens automatically when used with slot="end"

Values can optionally be contained in a time or data tag.
