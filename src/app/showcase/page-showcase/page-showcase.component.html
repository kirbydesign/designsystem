Page is a container element for a Kirby page. There should only be one kirby-page element per page. The page component gives you the following:

  • Scrollable page title and actions
  • Back navigation
  • Default markup for page title
  • Spacing of content

Title Alignment & Custom Toolbar Title

Titles are default left aligned but can be centered (titleAlignment="center") or right aligned (titleAlignment="right").

It also possible to supply a custom toolbar title (toolbarTitle="A Different Title") that is shown when the page title is scrolled above the toolbar.

If the toolbarTitle property is specified instead of title, the page title and actions will only be shown in the toolbar and not on the page itself.
