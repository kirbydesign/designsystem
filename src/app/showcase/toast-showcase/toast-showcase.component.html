To show the toast, create a ToastConfig and pass it to toastController.showToast const config: ToastConfig = 
{{ '{' }}
  message: 'Your toast message',
  messageType: 'success';
  durationInMs: 5000,
{{ '}' }};
this.toastController.showToast(config);
(Optional) if you need to know when the toast is dismissed, you can pass a callback function: this.toastController.showToast(config, myCallback); where myCallback can be e.g.: myCallback() {{ '{' }} ... {{ '}' }}
