Properties:

Name Description Input values
imageSrc Points to the src of the image location. string
altText Must be filled out - its the alt text attribute that screenreaders use when 'viewing' the image. string
size Yeah, as the name implies, this sets it size :) small, medium, large, xlarge
overlay Adds a dimmed overlay on the avatar. Useful when using the avatar on a white background and the image also has a white background. boolean
shadow Adds a box-shadow to the avatar boolean
