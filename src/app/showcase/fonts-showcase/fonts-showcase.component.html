Font sizes

Font sizes can be set by either using the font classes, e.g. .kirby-text-large, or by using the tags like h1, h2 & h3.

H3 | .kirby-text-medium

SCSS Function

If you need a value from the font-size map directly, you can use the font-size(value) function: 
.some-class {{'{'}}
  font-size: font-size('l');
{{'}'}}
