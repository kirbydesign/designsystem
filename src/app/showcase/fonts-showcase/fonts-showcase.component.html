The font sizes can be set by either using the font classes, e.g. kirby-text-large, or by using the right tags like h1, h2 & h3

Font can also be set with the font-size function directly on your own classes, e.g:

  .my-own-class {{'{'}}
    font-size: font-size("m");
    ...
  {{'}'}}

h1 - Heading 1

font-size: font-size("l"); font-weight: font-weight("bold"); CLASS: .h1 or .kirby-text-large

h2 - Heading 2

font-size: font-size("m"); font-weight: font-weight("bold"); CLASS: kirby-text-medium

h3 - Heading 3

font-size: font-size("n"); font-weight: font-weight("bold"); CLASS: kirby-text-normal-bold

p - Paragraph (Default body font)

font-size: font-size("n"); CLASS: kirby-text-normal

Small text

font-size: font-size("s"); CLASS: .sm or kirby-text-small

Small text - light

font-size: font-size("s)""; font-weight: font-weight("light"); CLASS: .sm-light or kirby-text-small-light

x-small text

font-size: font-size("xs"); CLASS: .xs or kirby-text-xsmall

xx-small text

font-size: font-size("xxs); CLASS: .xxs or kirby-text-xxsmall
